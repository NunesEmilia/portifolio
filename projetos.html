Spero Game

Game desenvolvido no primeiro semestre do curso de sistemas e mídias digitais, como parte do projeto integrado. O objetivo era desenvolver um jogo de plataforma com uma temática sócio-ambiental. Nele o jogador controlava um robô que tinha por objetivo desativar uma fabrica puluidora.

Movie Maker

Recurso desenvolvido como projeto integrado do segundo semestre. O objetivo desse aplicativo era auxiliar os alunos iniciantes do curso de sistema e midias digitais a desenvolverem produções audiovisuais. O aplicativo teria instruções sobre técnicas de criação de roteiros até boas praticas em captação de áudio.

All Type

Cartaz desenvolvido durante a disciplina de comunicação visual I. O objetivo era desenvolver um anuncio All Type. Um anúncio alltype é caracterizado pelo uso somente de letras, com criatividade de seus criadores para ir além de escrever um bom texto.
